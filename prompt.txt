buatkan table session dengan kolom:
- id integer autoincrement primary key
- token varchar(255) not null (isinya UUID untuk token yang login)
- user_id integer not null (FK ke table users)
- created_at timestamp default current_timestamp
- updated_at timestamp default current_timestamp on update current_timestamp

buatkan api untuk regestrasi user login user

request body : {
    "username": "rakhman",
    "password": "12345678"
}
untuk password harus di hash menggunakan bcrypt

response body (success): 
{
    "status": 200,
    "message": "User login successfully",
    "data": "token"
}

response body (error):
{
    "status": 401,
    "message": "User not found",
    "data": null
}

Struktur folder didalam src
- route : ini berisi routing ElysiaJS
- service : berisi logic bisnis aplikasi

Struktur penamaan file
- route : nama file harus diawali dengan "route-" contoh : route-user.ts
- service : nama file harus diawali dengan "service-" contoh : service-user.ts
