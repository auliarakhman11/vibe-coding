Struktur folder didalam src
- route : ini berisi routing ElysiaJS
- service : berisi logic bisnis aplikasi

Struktur penamaan file
- route : nama file harus diawali dengan "route-" contoh : route-user.ts
- service : nama file harus diawali dengan "service-" contoh : service-user.ts
